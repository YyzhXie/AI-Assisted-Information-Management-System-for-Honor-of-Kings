## TC13 连续登录多个玩家账号

功能：连续同角色登录稳定性  
输入：`p001/123456` 登录后登出，继续 `p002/123456` 登录后登出，继续 `p006/123456` 登录后登出  
预期输出：每次都进入玩家菜单，并显示对应玩家昵称  
实际输出：依次显示“欢迎 阿离同学”、“欢迎 峡谷小周”、“欢迎 法术核心”  
结果：通过  
发现的 bug：无

## TC14 连续登录多个管理员账号

功能：连续同角色登录稳定性  
输入：`admin/admin123` 登录后登出，继续 `coach/coach123` 登录后登出  
预期输出：每次都进入管理员菜单，并显示对应管理员名称  
实际输出：依次显示“欢迎 系统管理员”、“欢迎 战术教练”  
结果：通过  
发现的 bug：原数据集只有一个管理员，已补充 `coach/coach123` 用于覆盖该场景
